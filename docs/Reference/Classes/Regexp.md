# Regexp

```code
(Regexp) -> regexp
```code

### :compile

```code
(. regexp :compile pattern) -> :nil | meta
```code

### :match?

```code
(. regexp :match? text pattern &optional meta) -> :t | :nil
```code

### :search

```code
(. regexp :search text pattern &optional meta) -> (matches submatches)
```code
