# Emap

```code
(Emap [num_buckets]) -> emap
```code

### :copy

```code
(. emap :copy) -> emap
```code

### :deep_copy

```code
(. emap :deep_copy) -> emap
```code

### :empty

```code
(. emap :empty) -> emap
```code

### :erase

```code
(. emap :erase key) -> emap
```code

### :find

```code
(. emap :find key) -> :nil | val
```code

### :insert

```code
(. emap :insert key val) -> emap
```code

### :move

```code
(. emap :move) -> emap
```code

### :resize

```code
(. emap :resize num_buckets) -> emap
```code
