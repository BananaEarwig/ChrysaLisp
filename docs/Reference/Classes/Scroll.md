# Scroll

## View

```code
(Scroll flags) -> scroll
```code

### :action

```code
(. scroll :action data) -> scroll
```code

### :add_child

```code
(. scroll :add_child child) -> scroll
```code

### :layout

```code
(. scroll :layout) -> scroll
```code

### :mouse_wheel

```code
(. scroll :mouse_wheel event) -> scroll
```code

### :pref_size

```code
(. scroll :pref_size) -> (width height)
```code
