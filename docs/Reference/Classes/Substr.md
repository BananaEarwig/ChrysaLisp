# Substr

```code
(Substr) -> substr
```code

### :compile

```code
(. substr :compile pattern) -> :nil | meta
```code

### :match?

```code
(. substr :match? text pattern &optional meta) -> :t | :nil
```code

### :search

```code
(. substr :search text pattern &optional meta) -> (matches submatches)
```code
